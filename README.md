# Car Race Game

シンプルな2Dカーレースゲームです。プレイヤーは自車を左右に操作して、前方から迫る敵車を避けつつできるだけ長く走行します。

## ゲーム概要

- 画面中央の路面を走行するカーレースゲーム
- 左右キーで自車を操作
- 正面から複数の敵車が出現し、時間経過で敵車の数が増加
- アイテムを取得することで特殊効果を発動
- 路面に氷や泥のパッチがあり、接触すると操作に影響が発生
- ライフが0になるとゲームオーバー
- 走行距離に応じてスコアが増加

## 操作方法

- 左矢印キー: 左移動
- 右矢印キー: 右移動

## アイテム効果

- 無敵アイテム: 一定時間ダメージ無効
- 縮小アイテム: 自車が小さくなり回避しやすくなる
- 1UPアイテム: ライフを1回復

## 実行方法

以下のコマンドでコンパイルと実行ができます。raylib がインストールされている必要があります。

```bash
gcc game10.c -o game10 -lraylib -lGL -lm -lpthread -ldl -lrt -lX11
./game10
```

## ファイル

- `game10.c` - ゲーム本体のソースコード
- `game10` - コンパイル済み実行ファイル
- `assets/` - ゲームで使用する画像ファイル
  - `player.png`
  - `enemy.png`
  - `item_invincible.png`
  - `item_shrink.png`
  - `item_life.png`

## 注意事項

- 実行には raylib と OpenGL の動作環境が必要です
- 画面ウィンドウを閉じるとゲームを終了します
